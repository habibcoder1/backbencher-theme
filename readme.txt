== Description ==

This is a Technology Company based theme. You can make any type of Digital Marketing, It Related Website with this template
Backbencher Studio is a super-fast, seo-friendly, responsive and browser-compatible theme. It is free theme, so you can use it in multiple websites.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Backbencher Studio in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
